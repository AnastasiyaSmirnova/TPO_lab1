Описание доменной модели
1. 2 вида существ, базирующихся на классе "Человек"
    - человек обычный - UsualHuman
    - видоизмененный - MutateHuman

    Общее для всех =
    имя (идентификатор),
    эмоциональное состояние,
    описание тела: положение - можно сидеть\стоять\идти,
    число голов: для обычного всегда 1, для мутанта от 1 и больше + для мутантов описание занятости головы - изначально все свободны,
    показатель собранности ~ внимания: сосредоточен\расслаблен

2. Для человека обычного характерно привычное нам строение, но некоторые особи могут владеть уникальной способностью:
    увеличивать число предметов, на которые они смотрят (вот так бы с конфетами или деньгами в жизни) - каждый взляд + 100

    Поэтому для обычного человека нужно указать есть ли уникальные способности, default - no
    Простой человек сначала стоит, рассредоточенный, но уже нервный

3. Для мутанта характерно изменяемое число голов не меньше 1
В зависимости от числа голов раздаются приоритеты (чем больше голов, выше приоритет). Высокий приоритет позволяет выполнять более
ответственные задачи.
Для каждой головы можно устанавливать активности (если активности есть => isHeadFree = false)
Активности можно удалять и вообще чистить лист активностей для головы.
